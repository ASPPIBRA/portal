You can add a beta badge to any tab by adding the value of the tab to the `betaBadgeValueIds` array on the ICTabs
parent component. Make sure you use ICTabs for the parent component. You can use either ICTabItem or the TabItem
component provided by Docusaurus; they are the same component. The value provided to `betaBadgeValueIds` should match the value provided to the TabItem value prop.
