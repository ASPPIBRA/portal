## Overview

A Bitcoin address is a hash value. There are currently two types of Bitcoin address types, P2SH and P2PKH. An address's full public key cannot be provided when sending transactions and token transfers. To redeem transactions sent to your Bitcoin address, you must provide both the transaction signature and public key used to sign the transaction.
