## How to submit a proposal
To submit a proposal, it is recommended to use a neuron that can be controlled with a command line tool.
For example, you can review concrete instructions how to [submit a proposal of type `Motion`](https://wiki.internetcomputer.org/wiki/How-To:_Create_an_NNS_motion_proposal).
